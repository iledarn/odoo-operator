Features

Odoo-Operator fully leverages Red Hat’s operator framework to run Odoo instances at scale atop a cloud or on-premise kubernetes cluster (= open standard based Odoo.sh alternative).

Compliant with 12-Factor

Don’t keep state within the Odoo instance enabeling fully horizontal scalability through an external session store and a shared filestore.

Cluster Deployment

Let’s you deploy one or several code versions into a namespaced cluster with a database namesapce, and quotas attached to it. This is greate for custom project with dedicated resources.

Instance Deployment & Seeding

Let’s you seed and deploy a database instance with optionally a database and filestore quota attached.

Quotas

Set per database and per cluster quotas for full control of your provisioned resources.

Backups

Create automatic and manual incremental snapshots or packaged backups for your database and filestore.

Migration

Create migration instances and jobs to migrate your databases automatically, fully integrated with the Odoo migration API.
